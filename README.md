## 📊 Visualization
- **Treemap:**  
  - Displays hierarchical data as nested rectangles.  
  - Each rectangle size corresponds to the department’s budget.  
  - Useful for **budget distribution analysis** and quick comparison.  

- **Dendrogram:**  
  - Represents hierarchical clustering of data points.  
  - Shows how items are grouped step by step into clusters.  
  - Useful for **pattern recognition** and **classification tasks**.

## 📈 Output & Interpretation
- The **treemap** clearly shows how budgets are distributed across departments, making it easy to identify which department has the largest or smallest allocation.  
- The **dendrogram** illustrates hierarchical relationships among data points, helping in understanding similarity and grouping.  
- Together, these visualizations demonstrate how **interactive and specialized charts** can be applied to real‑world problems.
